# AuraLite 0.1.0 Beta

AuraLite is a lightweight aura tracker for WoW Retail focused on fast setup, clean visuals, cast-driven timers, and a workflow familiar to WeakAuras users.

## Highlights

- Modern editor with live preview
- Stable aura identities and position persistence
- Group movers and shared group layout
- Synthetic proc/consume rules
- Stackable and extend-to-cap timer logic
- Import/export for single auras and groups
- Visual bar texture picker and improved appearance controls

## Recommended Beta Testing

- Standalone aura positioning after save/reload
- Group movement and shared layout
- Proc auras with consume rules
- Stack-based buffs
- Import/export round-trips

## Notes

- This is a beta build intended for focused user testing.
- If you find a bug, please include:
  - the aura or group involved
  - what action triggered the issue
  - whether it happened after save, import, reload, or combat
