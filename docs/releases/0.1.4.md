# AuraLite 0.1.4

## Highlights

- Refined the new movers workflow with quick snap, reset, and lock actions.
- Made import/export and group assignment more visible directly inside the studio.

## Improvements

- Added a compact movers utility with `Lock and Close`, `Reopen Studio`, `Reset Visible`, and snap controls.
- Added more visible `Import` and `Export` actions in the main studio toolbar.
- Improved group assignment UX with clearer drag hints and direct drag & drop onto group rows.
- Added a lightweight warnings summary in the studio header.

## Fixes

- Fixed several studio layout regressions introduced during the movers/UI polish pass.
- Fixed advanced editor field overlap for checkboxes and spell list inputs.
- Fixed preview and panel spacing issues that caused text and controls to overlap.
- Fixed editor scroll content drawing past the bottom divider/footer.
- Restored better visibility and spacing for the live preview area.

## UI Polish

- Rebalanced studio column widths to give the configuration editor and preview proper space again.
- Improved wording and discoverability around moving auras into groups.
