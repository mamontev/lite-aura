# AuraLite 0.1.5

## Highlights

- Removed automatic default/sample group seeding so the addon no longer recreates demo groups on its own.
- Improved studio clarity with a better empty state, clearer drag-to-group guidance, and a cleaner default list view.

## Improvements

- `Not Loaded` sections now start collapsed by default to reduce visual noise.
- Added a richer empty state in the aura list with quick actions for `New Aura`, `Import`, and `Groups`.
- Made group assignment clearer by explicitly guiding users to drag auras from the library into groups on the left.
- Added drag hint feedback directly in the groups browser and group studio.

## Fixes

- Fixed sample/default groups reappearing after reload by removing them from empty profile defaults and cleanup migrations.
- Fixed drag-and-drop assignment directly onto groups in the left column.
- Fixed `Test Trigger` so it uses the same preview pipeline as the live editor preview.
- Fixed multiple Group Studio spacing and overlap issues in the header and group list area.

## UI Polish

- Refined wording around group management and aura assignment.
- Kept the studio focused on real user content instead of pre-seeded demo structures.
