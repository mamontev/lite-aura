# AuraLite 0.1.3

## Highlights

- Added a compact movers window so you can reposition auras without the full studio covering the screen.
- Reduced the main studio size to fit more comfortably on smaller displays.

## Fixes

- When `Movers` is enabled, the full studio now hides and is replaced by a small draggable utility window.
- Added quick actions to lock positions immediately or reopen the studio while keeping movers available.
- Improved the editing flow for users whose aura icons were blocked behind the settings window.

## UI Polish

- Tightened the main studio footprint from `1700x920` to `1520x820`.
- Kept the movers workflow focused on placement instead of forcing the full editor to stay open.
